# Complete example

Creates an AWS IAM role for secure, keyless authentication from GitHub Actions using OpenID Connect (OIDC)
with ECR and S3 permissions.

## usage

```
terraform init
terraform plan
terraform apply
```

Note that this example may create resources which cost money. Run `terraform destroy` to destroy those resources.
